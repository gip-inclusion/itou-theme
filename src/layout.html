Utilities for Flex layout

Documentation
Display

Use our display utilities for responsively toggling common values of the display property. Mix it with our grid system, content, or components to show or hide them across specific viewports.

Flexbox

Should you need to add display: flex to an element, do so with .d-flex or one of the responsive variants. You’ll need this class or display value to allow the use of our extra flexbox utilities for sizing, alignment, spacing, and more.

Visibility

When toggling display isn’t needed, you can toggle the visibility of an element with our visibility utilities. Invisible elements will still affect the layout of the page, but are visually hidden from visitors.

Spacing

Documentation
Bootstrap comprend une large gamme de classes utilitaires de margin et de padding pour modifier l'espacement d'un élément.

Attribuez des valeurs de margin ou de padding adaptées à un élément ou à un ensemble d'éléménts.

Les classes sont nommées en utilisant le format {property}{sides}-{size} pour xs et {property}{sides}-{breakpoint}-{size} pour sm, md, lg, et xl.

Par exemple, les tailles sont l'une des suivantes pour les margin-top :

  • mt-0 - pour la class qui supprime margin-top en le réglant sur 0
  • mt-1 - pour la class qui définit margin-top avec la taille $spacer * .25 (soit 4px)
  • mt-2 - pour la class qui définit margin-top avec la taille $spacer * .5 (soit 8px)
  • mt-3 - pour la class qui définit margin-top avec la taille $spacer (soit 16px)
  • mt-4 - pour la class qui définit margin-top avec la taille $spacer * 1.5 (soit 24px)
  • mt-5 - pour la class qui définit margin-top avec la taille $spacer * 2 (soit 32px)
  • mt-6 - pour la class qui définit margin-top avec la taille $spacer * 3 (soit 48px)
  • mt-7 - pour la class qui définit margin-top avec la taille $spacer * 4 (soit 64px)
  • mt-8 - pour la class qui définit margin-top avec la taille $spacer * 5 (soit 80px)
  • mt-9 - pour la class qui définit margin-top avec la taille $spacer * 6 (soit 96px)
  • mt-auto - pour la class qui définit margin-top a auto
