Comment utiliser le bouton Mon portail + menu Nexus

  • Copier le contenu du DOM HTML <div class="nx-dropdown">
  • Coller le contenu du DOM HTML <div class="nx-dropdown"> dans votre <div class="fr-header__tools-links">
  • Copier/coller l'ensemble des styles présents dans la balise <style> du code source de cette page HTML
  • Remplacer l'url du svg https://cdn.jsdelivr.net/npm/@gouvfr/dsfr@1.14.3/dist/icons/arrows/arrow-down-s-line.svg par celui présent sur votre site
  • Télécharger les images de logo des services (y compris Nexus dans le bouton), uploader sur votre site et modifier les urls d'utilisation de ces images
