Lottie

Pour utiliser Lottie il suffit d'ajouter l'appel aux cdn du js <script src="https://unpkg.com/@dotlottie/player-component@latest/dist/dotlottie-player.mjs" type="module"></script> puis de copier le code nécessaire à la création du Web Components en spécifiant l'url vers le fichier d'animation *.json.

Par exemple : 

<dotlottie-player src="https://lottie.host/272b60dd-462d-42a3-8ed6-fec4143633d6/X4FxBascRI.json" background="transparent" speed="1" style="width: 600px; height: 600px" class="img-fluid" direction="1" mode="normal" loop autoplay></dotlottie-player>
