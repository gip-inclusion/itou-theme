# Documentation du thème Itou

Bienvenue sur le Storybook du **theme-itou**.

Cette documentation présente les composants, les styles et les patterns utilisés dans l’écosystème de la plateforme de l’inclusion.
Le design system des Emplois de l'inclusion est créé avec Figma.
Sa transposition front-end est construite sur la base de [Bootstrap 5.3](https://getbootstrap.com/docs/5.3/getting-started/introduction/). La personnalisation est faite avec l'adaptation des variables, la customisation des composants et la création d'éléments spécifiques.
Les classes standards de Bootstrap respectent le naming de Bootstrap 5.3. Les classes et composants custom du thème des Emplois de l'inclusion suivent les conventions du BEM naming.


## Ce que vous trouverez ici

- Des composants UI réutilisables (boutons, alertes, badges, accordéon, etc.)
- Des exemples d’usage avec le HTML rendu
- Les variantes disponibles et les bonnes pratiques d’intégration

## Objectif

Ce Storybook sert de référence visuelle et technique pour faciliter l’implémentation du design system dans les différentes plateformes consommatrices.

## Accessibilité

Les composants documentés ici doivent respecter les exigences d’accessibilité (RGAA), notamment :

- structure sémantique
- navigation clavier
- contrastes suffisants
- libellés explicites pour les éléments interactifs
