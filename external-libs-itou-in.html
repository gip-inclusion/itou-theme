Intro JS

Pour utiliser Intro JS il suffit d'ajouter l'appel aux cdn du js <script src="https://cdn.jsdelivr.net/npm/intro.js@7.2.0/intro.min.js"></script> puis instancier introJs.tour()

Par exemple : 
                    
  <script>
  introJs.tour().setOptions({
    dontShowAgainCookie: 'introjsPass-dontShowAgain',
    dontShowAgain: true,
    showProgress: false,
    showBullets: true,
    skipLabel: "",
    nextLabel: "Suivant",
    prevLabel: "Précedent",
    doneLabel: "Fin",
    dontShowAgainLabel: "Ne plus afficher",
    steps: [{
        title: 'Bonjour 👋',
        intro: 'Découvrez votre nouvelle box de PASS IAE'
      },
      {
        element: document.querySelector('#introjsPass01'),
        title: 'Votre PASS est valide',
        intro: 'Ici l\'etat actuel de votre PASS est indiqué'
      },
      {
        element: document.querySelector('#introjsPass02'),
        title: 'Valide jusqu\'a cette date'
      },
      {
        element: document.querySelector('#introjsPass03'),
        intro: 'Vous pouvez l\'afficher en cliquant ici'
      }]
    }).start();
</script>

Les styles seront automatiquement chargés par le thème itou, il est donc inutile d'inclure les styles par défaut.
